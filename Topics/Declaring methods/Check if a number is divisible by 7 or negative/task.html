Check if a number is divisible by 7 or negative

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Given an integer number, we need to find out whether it is divisible by 7. If the number is divisible by 7, return the string 'YES'. If not, return 'NO'. Additionally, if the input number is a negative value, return 'INVALID'. Input: 16 Output: NO Input: -1 Output: INVALID Write a method that takes in an integer as its parameter to perform this task. This should then return a corresponding string based on the conditions given above.
Sample Input:
5

Sample Output:
NO


Sample Input:
7

Sample Output:
YES
